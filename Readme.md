### Homework 62
Написати функцію generateList(array), яка приймає масив із чисел та масивів чисел (наприклад [1,2,3]) і генерує список із елементів:

Copy code
<ul>
	<li>1</li>
	<li>2</li>
	<li>3</li>
</ul>
Якщо ж у масиві зустрічається масив (наприклад, [1,2, [1.1,1.2,1.3] ,3]) то робити вкладений список. Для перевірки масиву використовуйте Array.isArray()

Copy code
<ul>
	<li>1</li>
	<li>2</li>
	<li>
		<ul>
			<li>1.1</li>
			<li>1.2</li>
			<li>1.3</li>
		</ul>
	</li>
	<li>3</li>
</ul>



пояснення
const array = [1, 2, [1.1, 1.2, 1.3], 3];: Це оголошення масиву array, який містить числа і один вкладений масив.
const generateList = (arr) => { ... }: Це оголошення функції generateList, яка приймає один аргумент arr. arr очікується бути масивом.
const list = document.createElement('ul');: Створює новий елемент ul (невпорядкований список) і зберігає його в змінній list.
for (let i = 0; i < arr.length; i++) { ... }: Це цикл for, який проходиться по кожному елементу масиву arr.
const listItem = document.createElement('li');: Усередині циклу створюється новий елемент li (елемент списку) для кожного елемента масиву. Цей елемент зберігається в змінній listItem.
const node = document.createTextNode(arr[i]);: Створює новий текстовий вузол зі значенням поточного елемента масиву (arr[i]). Цей текстовий вузол зберігається в змінній node.
listItem.append(node);: Додає текстовий вузол до елемента li. Це означає, що текстовий вузол тепер є дочірнім елементом li.
list.appendChild(listItem);: Додає елемент li до елемента ul. Це означає, що елемент li тепер є дочірнім елементом ul.
if (Array.isArray(arr[i])) { ... }: Це умовний оператор, який перевіряє, чи є поточний елемент масиву (arr[i]) масивом.
listItem.appendChild(generateList(arr[i]));: Якщо arr[i] є масивом, цей рядок коду викликає функцію generateList з arr[i] як аргументом і додає повернений список ul до поточного елемента li.
listItem.removeChild(listItem.firstChild);: Після додавання вкладеного списку цей рядок коду видаляє перший дочірній вузол з елемента li, який є текстовим вузлом. Це залишає вкладений список ul, але видаляє текстовий вміст.
return list;: Після завершення циклу, функція повертає елемент ul, який тепер містить елемент li для кожного елемента масиву arr. Кожен елемент li містить текстовий вузол зі значенням відповідного елемента масиву, або вкладений список, якщо елемент масиву був масивом.
console.log(generateList(array));: Цей рядок коду викликає функцію generateList з масивом array як аргументом і виводить результат в консоль.
