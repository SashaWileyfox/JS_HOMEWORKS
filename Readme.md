### Homework #24
task #1
Даний масив із числами. За допомогою циклу знайдіть суму елементів цього масиву, виведіть її в консоль.

const arr = [1, 2, 3, 4, 5];
Коли ви використовуєте a <= arr.length в умові циклу, цикл спробує отримати доступ до arr[arr.length], який не існує, і поверне undefined. Це може призвести до непередбачуваних результатів або помилок у вашому коді.

Тому, коли ви ітеруєте масив в JavaScript, вам слід використовувати a < arr.length в умові циклу, щоб уникнути виходу за межі масиву. З цією умовою цикл пройде всі елементи масиву від arr[0] до arr[arr.length - 1], що є останнім елементом масиву.
