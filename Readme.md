### Homework #6
Отримати від користувача 3 рядки та вивести їх у довільному порядку однією командою (конкатенація);
Розбити за цифрами п'ятизначне число і вивести у вихідному порядку через пробіл.

Рішення:
Змінна number: створили змінну num і присвоїли їй значення 12345.
Змінна digits: створили порожній масив digits, який буде використовуватися для збереження окремих цифр числа.
Змінна number: перетворили число num на рядок, використовуючи метод .toString(). Тепер number містить рядок "12345".
Метод .split(''): Ви використали метод .split(''), щоб розбити рядок sNumber на окремі символи (цифри). Результатом є масив ["1", "2", "3", "4", "5"].
Метод .map(Number): Ви використали метод .map(Number), щоб перетворити кожний символ (цифру) з масиву на число. Тепер digits містить масив чисел [1, 2, 3, 4, 5].
Метод .join(' '): На останньому кроці ви використовуєте .join(' '), щоб об’єднати всі числа в рядок, розділені пробілами. Результат виведеться у вигляді "1 2 3 4 5".
