Змінна commentButtonInitialized: Це логічна змінна, яка вказує на те, чи вже був ініціалізований івент-слухач для кнопки "Get Comments". Початково вона має значення false, що означає, що івент-слухач ще не був доданий.

Перевірка if (!commentButtonInitialized): Ця умова перевіряє, чи commentButtonInitialized є false. Це означає, що івент-слухач ще не був доданий. Якщо умова виконується (тобто commentButtonInitialized є false), то відбувається наступне:

Отримується посилання на кнопку "Get Comments" за допомогою document.querySelector('[data-comments-button]').
Перевіряється, чи commentButton не є null або undefined.
Якщо кнопка існує (commentButton не є null), до неї додається івент-слухач на подію click, який викликає функцію getComments(post.id).
Після додавання івент-слухача значення commentButtonInitialized змінюється на true, щоб в майбутньому уникнути повторного додавання івент-слухача.
Забезпечення одноразовості додавання івент-слухача: Цей механізм гарантує, що івент-слухач буде додано лише один раз при першому виклику handleCommentButtonClick, а потім вже не буде додаватися при наступних викликах. Це дозволяє уникнути накопичення багатьох івент-слухачів і зайвого споживання ресурсів.
