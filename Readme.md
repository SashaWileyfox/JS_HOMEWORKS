 #task #2
Розбити за цифрами п'ятизначне число і вивести у вихідному порядку через пробіл.

Рішення:
Змінна number: створила змінну num і присвоїли їй значення 12345.
Змінна digits: створила порожній масив digits, який буде використовуватися для збереження окремих цифр числа.
Змінна number: перетворила число num на рядок, використовуючи метод .toString(). Тепер number містить рядок "12345".
Метод .split(''): (розбиває рядок на масив рядків шляхом розділення рядка вказаним підрядком), щоб розбити рядок sNumber на окремі символи (цифри). Результатом є масив ["1", "2", "3", "4", "5"].
Метод .map(Number): (трансформу' один масив в інший), щоб перетворити кожний символ (цифру) з масиву на число. Тепер digits містить масив чисел [1, 2, 3, 4, 5].
Метод .join(' '): (об'єднює елементи масиву в рядок з вказаним розділювачем) На останньому кроці використову. .join(' '), щоб об’єднати всі числа в рядок, розділені пробілами. Результат виведеться у вигляді "1 2 3 4 5".
